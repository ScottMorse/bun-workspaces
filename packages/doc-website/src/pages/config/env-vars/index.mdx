## Settable Environment Variables

Set these environment variables to override default values used. The [root configuration](/config/root) will take precedence over environment variables, 
and any values set explicitly in CLI or API arguments will take precedence over both.

<ul className="env-vars-list">
{Object.values(ENV_VARS_METADATA).map(envVar => <div key={envVar.envVarName}>
  <li className="env-var-item" key={envVar.envVarName}>
    <code>{envVar.envVarName}</code>
    <div>{envVar.description}</div>
    <div>If <code>{envVar.rootConfigDefaultsKey}</code> is set in the <Link href="/config/root" className="inline-link">root configuration</Link>, it will take precedence.</div>
  </li>
</div>)}
</ul>
