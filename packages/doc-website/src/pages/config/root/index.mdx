## Project Root Configuration

The root configuration is the general configuration for a [project](/concepts/glossary#project). It is read from a file in the project root directory or the root `package.json` file.

The values in <code>{CONFIG_DEFAULTS_KEY}</code> can also be set by [environment variables](/config/env-vars). Defaults provided in the configuration take precedence over environment variables.

See the [parallel max value](/concepts/parallel-scripts#parallel-max-value) and [shell option for inline scripts](/concepts/inline-scripts#shell) for information on the use of the config values.
