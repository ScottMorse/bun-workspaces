## Workspace Configuration

Configuration for a workspace is defined in a file in its directory or in its `package.json` file.

See [workspace aliases](/concepts/workspace-aliases) and [script execution order](/concepts/script-execution-order) for more information on the fields available.

### Type Definition

<SyntaxHighlighter language="typescript">
  {WORKSPACE_CONFIG_TYPE}
</SyntaxHighlighter>
