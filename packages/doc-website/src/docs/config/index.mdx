# Config File (bw.json)

You can create a config file at `bw.json` in your project root,
or you can pass a config file to the CLI with the `--config-file` (or `-c`) [option](/cli/index.html#global-option-configFile).

#### Example config

In this config, "app-a" is an alias for package "@my-org/application-a"
and "app-b" is an alias for package "@my-org/application-b".

The default
[log level](/cli/index.html#global-option-logLevel) can be set for the CLI as well.
